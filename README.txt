A Newsletter Plugin for WordPress. Design a template, send emails, people can subscribe through your website, double opt in option, track sent emails.

== Description ==

Ultimate Newsletter is a fully featured newsletter plugin, that gives you the power of a system like MailChimp inside of WordPress.

[Demo](http://demo.yendif.com/wordpress/ultimate-newsletter) | [Documentation](https://yendif.com/wordpress/ultimate-newsletter/getting-started.html) | [Support](https://yendif.com/forum/ultimate-newsletter-wordpress.html)

https://www.youtube.com/watch?v=PsiD9lYcBTg

= Newsletter Features =

* No HTML knowledge required, enjoy the Newsletter creation with our drag and drop editor
* Real time send process using a queue system with throttling enabling you to overcome ANY server limitation!
* Customizable subscription widget
* Handle the subscription of registered users and visitors
* CAN-SPAM compliant
* Validation and Confirmation of the email address
* Automatic synchronization with your WordPress Users
* Handle unlimited subscribers, email-groups and newsletters
* Automatic inline-css conversion
* Preview and Send test newsletter to a specific email address
* Newsletter Statistics management (Number of Sent / Opened, Clicked)
* Handle SMTP Secured Connections (you can use GMail to send your newsletter)
* Administrator Notifications

== Installation ==

1. Download the plugin.
2. From the WordPress Admin Panel, click on Plugins => Add New.
3. Click on Upload, so you can directly upload your plugin zip file.
4. Use the browse button to select the plugin zip file that was downloaded, and then click on Install Now.
5. Once installed, click "Activate".

That's it! Visit the official [Documentation](https://yendif.com/wordpress/ultimate-newsletter/getting-started.html) page for further setup.

= Getting a 404 error? =
To get rid of the 404 error when viewing a download, you need to re-save your permalink structure. Go to Settings > Permalinks and click "Save Changes".
